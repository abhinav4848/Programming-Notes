# Google Search Operators

```
"exact query"
website.com: query
query 2017..2018
query ..2017 (so only 2017)
filetype:pdf query
query -excludedTerm
link:nytimes.com (to find pages that link here)
“Come * right now * me” (* is a wildcard placeholder that google will fill)
related:amazon.com
chocolate OR white chocolate
```
